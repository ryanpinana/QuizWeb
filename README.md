# QuizWeb

Nota per il funzionamento del DB:
Per fare funzionare il DB si necessita avere MongoDB e MongoDB Compass installato e avere una connessione sulla porta 27017 di nome QuizWeb, con all'interno un DB di nome quizdb, poi aggiungere all'interno di quizdb tre collezioni di nome questions, quizzes e users. Poi nelle colezioni importare i dati dai file JSON dalla cartella Database del progetto tramite il pulsante ADD DATA e poi Import JSON.
